Podemos criar tambem um texto marcado usando a tag <MARK>

Para mudar a cor da marcação é necessario usar CSS dentro da marcação do HTML. Como nesse exemplo agora. Utilizando ( STYLE BACKGROUND-COLOR!)

E no caso de um texto ter muitas marcaçoes, para nao ter que mudar tag por tag. Posso criar um seletor para tag (MARK) especifica no cabeçalho HEAD com a tag (STYLE BACKGROUDE-COLOR)
